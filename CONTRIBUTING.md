# Contributing to Token Tracker

感谢您对 Token Tracker 项目的兴趣！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题

如果您发现了 bug 或有功能建议，请：

1. 检查是否已有相似的问题报告
2. 创建一个新的 Issue，包含：
   - 清晰的标题
   - 详细的描述
   - 重现步骤（如果是 bug）
   - 环境信息（操作系统、Node.js 版本等）

### 提交代码

1. **Fork** 本仓库
2. 创建功能分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m 'Add some feature'`
4. 推送到分支：`git push origin feature/your-feature`
5. 提交 Pull Request

### 代码规范

- Python 代码遵循 PEP 8 规范
- JavaScript 代码使用 2 空格缩进
- 提交信息使用清晰的描述
- 添加必要的注释说明复杂逻辑

### 许可证

所有提交的代码都将采用 MIT 许可证发布。
